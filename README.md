# Omnifood Landing Page

Omnifood is a modern and responsive landing page for a food delivery subscription service. The website is designed using HTML, CSS, and JavaScript and demonstrates modern UI design principles, responsive layouts, and interactive web components.

## Project Overview

This project represents a landing page for a fictional company called **Omnifood**, which provides healthy meal subscriptions delivered to customers daily. The website highlights the service features, meal options, testimonials, pricing plans, and a signup form for users.

The design focuses on creating a **clean, attractive, and responsive user interface** that works smoothly across different devices such as desktops, tablets, and mobile phones.

## Features

- Responsive navigation bar with mobile menu
- Hero section with call-to-action buttons
- Featured section with company logos
- "How it Works" section explaining the service steps
- Meals section displaying available meal options
- Customer testimonials with image gallery
- Pricing plans for different subscriptions
- Call-to-action signup form
- Footer with contact information and useful links
- Smooth scrolling and interactive UI elements

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Functionality

- JavaScript is used to implement interactive elements such as **mobile navigation menu and smooth scrolling**.
- CSS **Flexbox and Grid** are used to create responsive layouts.
- Media queries are used to ensure the website adapts to different screen sizes.
- Hover effects and transitions improve the overall user experience.
